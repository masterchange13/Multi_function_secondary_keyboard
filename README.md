# Multi_function_secondary_keyboard

## 多功能辅助键盘 该键盘旨在辅助使用电脑，主要提供快捷键
## 下载地址 https://github.com/masterchange13/Multi_function_secondary_keyboard

## 开发平台
windows 11<br>

## 开发工具
arduino<br>
clion<br>

## 开发语言
c++

## 键盘功能
> 1-1 开机
> 1-2 关机
> 1-3 重启
> 1-4 锁屏
> 2-1 音量加
> 2-2 音量减
> 2-3 静音
> 2-4 music
> 3-1 下一首
> 3-2 上一首
> 3-3 播放/暂停
> 3-4
